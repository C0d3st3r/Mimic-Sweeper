Click X To Mark A Mimic
MIMIC-SWEEPER
DIFFICULT
INSTRUCTIONSYour mission, should you choose to accept it, is to find all the treasure in a dungeon, without being attacked by a mimic. Begin by clicking any grid. If the grid does not contain a mimic, it will either display how many mimics currently surround it or display treasure if no mimics are detected nearby. If you suspect a grid contains a mimic, click the X BEFORE clicking the grid in question. This marks the grid as dangerous. The game is over when every grid has either been cleared or correctly marked with a X. GOOD LUCK!
